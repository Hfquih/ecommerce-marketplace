# Ecommerce Marketplace

A full-stack e-commerce marketplace built with React, Vite, Node.js, Express, and MongoDB. The platform supports three user roles—customers, sellers, and administrators—providing a complete online shopping and product management experience.

Live App: https://ecommerce-marketplace-beryl.vercel.app

## Features

### Customer Features

* User registration and authentication
* Product browsing and advanced filtering
* Shopping cart management
* Secure checkout workflow
* Order tracking

### Seller Features

* Seller authentication and authorization
* Product creation, editing, and deletion
* Product image uploads via Cloudinary
* Order management

### Admin Features

* Administrative dashboard
* User management
* Product moderation
* Order oversight and analytics

## Technical Highlights

* JWT-based authentication and authorization
* Stripe payment integration
* Cloudinary image storage
* RESTful API architecture
* Input validation with Zod
* Security middleware including Helmet, rate limiting, and XSS protection
* MongoDB database with Mongoose
* Responsive React/Vite frontend
